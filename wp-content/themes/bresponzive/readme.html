'bresponZive' - News Magazine Responsive WordPress theme

'bresponZive' is a Fully Responisve theme with Magazine/blog style. It is a easy-to-customize and fully featured WordPress Theme. It has 2 columns, unlimited news sections. Theme is built on advanced solid framework which includes lots of advanced features and functions.
